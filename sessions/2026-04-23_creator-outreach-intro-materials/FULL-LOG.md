## Phase 1: 크리에이터 컨택 메시지 작성

나드리님 채널 정보를 위키에서 조회 → 위키에 없어 youtube-channel-research.md에서 확인함. 조회수 477K, 엑셀 자료 무료 나눔, 어필리에이트 적극 활동 확인. 인스타 DM 불가 확인 → 메일 컨택 방식 채택. 초기 "선물" 프레이밍 → 수익화 구조(쿠팡 파트너스 유사) 어필로 전환하여 메일 초안 작성 완료.

파파트래블: 인스타 팬심 인사 DM 발송 완료, "감사합니다" 답장 수신. 공구 진행 중인 크리에이터임을 확인 후 마이스토어 페이지 제안 DM 초안 작성. "비용 없음 + 초안 먼저 제작"으로 부담 최소화 방향 확정.

아재여행: 팬심 기반 첫 인사 메시지 발송 → 인스타 DM 제한으로 게시글 경유해 연결 완료. 다음 DM에서 마이스토어 제안으로 연결 예정. 아재여행의 콘텐츠 특성(코스/경비 꼼꼼 정리)과 연결된 메시지 방향 확정.

## Phase 2: 소개 자료 초안 및 방향 재설정

1차 소개 자료: 영업용 소개서(수익화 구조 중심) + 카드뉴스 제작. "너무 상업적"이라는 피드백 수용 → 실제 페이지 기능 자체를 보여주는 방향으로 전환.

주요 방향 변경:
- 실제 배포 페이지(mystore-mockup.vercel.app) 화면 직접 삽입
- Playwright로 메인/코스/경비 3개 화면 자동 스크린샷 캡처
- 크리에이터가 직접 제작한 폰 프레임 이미지(Frame 3/4/5.png) 적용
- iframe → 이미지로 교체하여 안정성 확보

## Phase 3: 소개 자료 완성

creator-intro.html 최종 구성:
- Hero: 테스트 중 뉘앙스 ("같이 해보실 분 찾고 있어요")
- WHY: 노션 번거로움 → 링크 흩어짐 → 제휴 수익 가능성
- FEATURES: 폰 프레임 이미지 3장으로 메인/코스/경비 설명
- CTA: 예시 페이지 보러가기 + 나만의 페이지 신청 버튼
- 신청 모달: 콘텐츠 링크(선택) + 인스타 아이디(필수) + 주의사항
- 폰 프레임 방식: 390px 모바일 전용 페이지

creator-cardnews.html 최종 구성:
- 6장 슬라이드 (표지 → WHY → 메인/코스/경비 화면 → CTA)
- 좌우 화살표 + 스와이프 + 키보드 이동 지원
- 폰 프레임 이미지 동일 적용
