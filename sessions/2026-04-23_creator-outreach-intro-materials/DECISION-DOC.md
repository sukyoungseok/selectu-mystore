# 크리에이터 컨택 전략 + 마이스토어 소개 자료 제작

날짜: 2026-04-23
주제: 크리에이터 개별 컨택 방식 확정 + 웹 소개서·카드뉴스 제작
유형: 실행 전략 + 결과물 제작
배경: 마이스토어 MVP 가설 검증을 위해 실제 크리에이터 컨택 및 소개 자료 제작이 필요함

---

## 최종 결정 사항

**1.** 나드리님 컨택 방식을 메일로 확정함. 인스타 DM 불가 + 어필리에이트 활동 적극적 확인 → 수익화 구조 어필 + 마이스토어 링크 포함 메일 초안 작성 완료

**2.** 파파트래블 컨택 방식 확정: 인스타 DM → 기존 공구 진행 중인 관계를 활용, 마이스토어 페이지 제안 DM 발송 예정

**3.** 아재여행 컨택 방식 확정: 팬심 인사 메시지 발송 후 DM 연결 완료 → 다음 DM에서 마이스토어 제안 예정

**4.** 소개 자료 방향 확정: 영업 멘트 제거, 실제 페이지 기능(코스/경비)을 직접 보여주는 방식 + 왜 만들었는지 스토리 포함

**5.** 소개 자료 2종 제작 완료: creator-intro.html(웹 소개서) + creator-cardnews.html(6장 카드뉴스)

**6.** 신청 폼 항목 2가지로 확정: 콘텐츠 링크(선택) + 인스타그램 아이디(필수)

**7.** 문의 이메일 marketing_partner@myrealtrip.com / 마케팅파트너팀으로 확정

**8.** GA4 로그 삽입 필요 확인 → 계정 생성 후 다음 세션에서 진행 예정

---

## 1. 배경 및 문제 정의

마이스토어 채널 페이지 9개 완성 후 실제 크리에이터 컨택 단계에 진입함. 크리에이터별로 인스타 DM 가능 여부가 달라 채널별 컨택 방식을 개별 확정해야 했음. 동시에 크리에이터에게 "마이스토어가 무엇인지" 설명하는 소개 자료가 없는 상태였음.

---

## 2. 크리에이터별 컨택 전략

| 크리에이터 | 채널 특성 | 컨택 방법 | 핵심 어필 포인트 |
|-----------|---------|---------|--------------|
| 나드리 | 조회수 477K, 엑셀 자료 무료 나눔, 어필리에이트 적극적 | 메일 | 수익화 구조(예약 수수료 셰어) |
| 파파트래블 | 공구 진행 중, 해외 콘텐츠 다수 | 인스타 DM | 마이스토어 페이지 제작 제안 |
| 아재여행 | 팬심 기반 첫 인사 완료, DM 연결됨 | 인스타 DM | 코스/경비 정리 + 수익화 |

컨택 메시지 핵심 프레이밍: "쿠팡 파트너스처럼 팔로워 예약 수수료 발생" + "비용 없이 초안 먼저 제작해드림"

---

## 3. 소개 자료 제작 방향 변경

초안 방향: 수익화 구조 중심 영업용 소개서 → 크리에이터로부터 "너무 상업적"이라는 피드백 수용

변경 방향:
- 노션 없이 코스/경비 정리 가능한 기능 자체를 보여줌
- 배포된 Vercel 페이지 스크린샷(Playwright 자동 캡처)을 직접 삽입
- 크리에이터가 직접 제작한 폰 프레임 이미지(Frame 3/4/5.png) 활용
- 왜 만들었는지 스토리 → 기능 소개 → 신청 폼 순서로 구성

---

## 4. 소개 자료 구성

### creator-intro.html (웹 소개서)
- Hero: "여행 크리에이터를 위한 새로운 수익화 방법을 테스트하고 있어요"
- WHY 섹션: 노션 번거로움 / 링크 흩어짐 / 제휴 수익 가능성
- FEATURES: 메인/코스/경비 화면 폰 프레임 이미지로 보여줌
- CTA 버튼 2개: 예시 페이지 보러가기 / 나만의 페이지 만들어보고 싶어요
- 신청 폼 모달: 콘텐츠 링크 + 인스타 아이디 입력
- 주의사항 명시: 테스트 단계, 신청해도 자동 제작 아님, 별도 연락 예정

### creator-cardnews.html (6장 카드뉴스)
- 슬라이드 1: 표지
- 슬라이드 2: 왜 만들었는지(WHY)
- 슬라이드 3~5: 메인/코스/경비 화면 폰 프레임으로 보여줌
- 슬라이드 6: CTA (테스트 함께할 분 모집)
