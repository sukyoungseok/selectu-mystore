# 크리에이터 컨택 전략 + 마이스토어 소개 자료 제작 - 회의록

날짜: 2026-04-23
참석: 석수경
주제: 크리에이터 개별 컨택 방식 확정 + 소개 자료 제작

---

## 안건

1. 크리에이터별 컨택 채널 및 메시지 방향 확정
2. 마이스토어 소개 자료(웹 소개서 + 카드뉴스) 제작
3. GA4 로그 삽입 필요성 확인

---

## 주요 논의

### 1. 나드리님 컨택

나드리님은 인스타 DM이 없어 메일로 컨택하기로 함. 위키 리서치에서 엑셀 자료 무료 나눔 + 어필리에이트 적극 활동 확인함. 초기 "선물" 프레이밍보다 수익화 구조(예약 수수료 셰어)를 직접 어필하는 방향으로 메일 초안 작성함.

### 2. 파파트래블 컨택

현재 공구 진행 중인 크리에이터로, 세일즈 성과 기대 가능함. 인스타 DM으로 감사 인사를 먼저 보낸 후 마이스토어 페이지 제안 DM으로 연결하는 2단계 방식 채택함.

### 3. 아재여행 컨택

팬심 기반 첫 인사 메시지를 게시글 댓글 통해 DM 연결함. 인스타 기본 DM 제한으로 인해 게시글 경유가 필요했음. 다음 DM에서 마이스토어 제안으로 자연스럽게 연결 예정.

### 4. 소개 자료 방향 재설정

초안이 "너무 상업적"이라는 피드백 수용. 수익화 구조 설명보다 실제 페이지 기능(코스/경비)을 직접 보여주는 방식으로 전환함. Playwright로 배포 페이지 스크린샷 자동 캡처 후 삽입하고, 크리에이터가 직접 제작한 폰 프레임 이미지로 교체함.

### 5. 신청 폼 및 안내사항

신청 폼 항목을 콘텐츠 링크(선택) + 인스타 아이디(필수) 2가지로 간소화함. 테스트 단계임을 명시하고, 신청 시 자동 제작이 아님을 안내하는 주의사항 문구를 폼 내부에 삽입함. 문의 이메일은 marketing_partner@myrealtrip.com으로 확정함.

### 6. GA4 로그

현재 채널 페이지에 로그가 없음을 확인함. GA4가 방문자 수 + 버튼 클릭 추적에 적합하다고 판단함. GA4 계정 생성은 구글 로그인이 필요해 수경님이 직접 진행해야 함.
