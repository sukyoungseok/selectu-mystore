- **날짜:** 2026-04-10
- **주제:** 마이스토어 여행 코스 목업 UX 방향 확정 및 YouTube 영상 분석 인프라 구축
- **유형:** 제품 기획 + 도구 설정
- **배경:** 마이스토어 프로젝트 영준님 보고 전 목업 퀄리티 개선 필요 및 크리에이터 콘텐츠 분석 자동화 방향 탐색

## 최종 결정 사항

**1.** 여행 코스 목업 UX 방식을 세로 타임라인 + 스팟별 인라인 상품 카드 구조로 확정함

**2.** 실제 크리에이터 게시글(@june_fairytale, 상하이 3박4일) 기반으로 목업 데이터를 업데이트함 — 방콕 가상 데이터 대신 실제 인스타그램 콘텐츠 활용

**3.** YouTube MCP(sfiorini/youtube-mcp)를 설치 완료함 — Claude 재시작 후 영상 자막 추출 기능 활용 가능

**4.** 목업 3가지 스타일(A 다크 / B 라이트 / C 그라데이션) 유지하되, 모두 동일한 타임라인+인라인 상품 구조 적용

---

## 1. 배경 및 문제 정의

마이스토어 프로젝트는 여행 크리에이터 전용 링크인바이오 서비스로, 여행 코스 섹션 내 MRT 상품 연결이 핵심 기능임. 기존 목업의 문제점은 다음과 같음:

- 상품이 코스 하단에 별도 섹션으로 분리되어 있어 눈에 띄지 않음
- 코스 흐름(타임라인)이 없어 여행 동선 파악이 어려움
- 방콕 가상 데이터 사용으로 실제 크리에이터 케이스와 거리가 있음

---

## 2. UX 개선 방향

### 2-1. 기존 구조의 문제

초기 목업(3가지 시도 모두)은 상품을 코스 하단에 모아서 나열하는 구조였음. 사용자 피드백: "상품이 눈에 안 띈다", "코스 흐름이 없다"

### 2-2. 채택한 구조

세로 타임라인 방식으로 코스 흐름을 표현하고, 관련 스팟 바로 아래에 예약 카드를 인라인으로 배치함. 예시:

- 공항 도착 스팟 → 바로 아래 항공권 예약 카드
- 디즈니랜드 스팟 → 바로 아래 입장권 예약 카드
- 황푸강 유람선 스팟 → 바로 아래 유람선 예약 카드

이 방식은 코스 맥락 안에서 자연스럽게 구매 의도를 유발하는 구조임.

### 2-3. 상품 카드 디자인

각 인라인 상품 카드 구성 요소:
- 이모지 아이콘 (카테고리 직관적 구분)
- 상품명 + 카테고리 설명
- 가격 (크고 색상 강조)
- 예약하기 → 버튼 (CTA 명확화)
