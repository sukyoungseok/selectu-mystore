# 마이스토어 파일럿 전략 피벗: 쿠폰페이지 + 마이링크 전환

_작성일: 2026-05-12 | 작성자: 석수경_

---

## 배경

크리에이터들에게 전용 마이스토어 페이지를 무료로 제작해드리는 방식으로 컨택했으나 두 가지 문제 확인됨.

- **크리에이터 측**: 수락은 했지만 실제로 활용하지 않음. 무보수 제안이다 보니 그들의 우선순위에서 밀림
- **운영 측**: 페이지 1개 제작에 시간 과다 소요. 결과 대비 리소스 비효율

## 전략 전환

| | 기존 | 신규 |
|--|------|------|
| 제작물 | 마이스토어 풀 페이지 (여행정보·경비·링크 모음) | 쿠폰페이지 + 마이링크 세트만 |
| 크리에이터 동기 | 없음 (무보수) | 예약 발생 시 수수료 수령 가능 |
| 제작 시간 | 과다 | 단축 (베리에이션 자동화 예정) |

### 용어 정리

- **쿠폰페이지**: 팔로워가 MRT 쿠폰팩을 받을 수 있는 프로모션 페이지. 크리에이터별로 발급.
- **마이링크**: 크리에이터 전용 MRT 제휴링크. 이 링크를 통한 예약 발생 시 수수료 지급.
- **마이스토어**: 기존에 제작하던 여행정보·경비·링크 모음 바이오링크 페이지.

### 실행 플로우

```
릴스 게시 (여행 일정표 공유)
  → "댓글 달면 쿠폰팩 증정" 안내
    → DM으로 쿠폰페이지 링크 전달
      → 팔로워가 쿠폰페이지에서 MRT 상품 예약
        → 크리에이터 수수료 발생
```

### 크리에이터 유형별 처우

| 유형 | 조건 | 제공 |
|------|------|------|
| 메가 인플루언서 3인 | 원고료 지급 | 마이스토어(기존 형태) + 쿠폰페이지 + 마이링크 |
| 일반 인플루언서 | 무보수 | 쿠폰페이지 + 마이링크 |
